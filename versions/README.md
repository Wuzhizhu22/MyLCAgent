# 版本管理说明

本目录用于存放项目学习过程中的不同版本实现，目的是记录学习轨迹和功能演进过程。

## 版本命名规范

采用功能描述+版本号的命名方式，便于清晰了解每个版本的特点：

- **基础版本**：`basic.py` - 包含最基础的智能体功能

## 版本演进记录

| 版本文件                   | 主要功能                                          | 学习重点                           |
| -------------------------- | ------------------------------------------------- | ---------------------------------- |
| basic.py                   | 基础智能体实现，支持工具调用和结构化响应          | LangChain 智能体创建、工具调用基础 |
| dynamic_model_selection.py | 实现动态模型选择，根据对话复杂度切换基础/高级模型 | 中间件使用、动态模型选择策略       |
